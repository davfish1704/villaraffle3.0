## Styling Guidelines

### Color Palette (Premium/Luxury)
```
Primary Gold/Champagne: #D4AF37 or #C9A961
Dark Background: #0A0A0A or #111111
White Text: #FFFFFF
Gray Text (secondary): #A0A0A0
Accent Green (trust): #10B981
```

### Typography
```
Headlines: Serif font (e.g., Playfair Display, Cormorant Garamond)
Body: Sans-serif (e.g., Inter, DM Sans)
Accent: Uppercase, wide letter-spacing for labels
```

### Spacing
```
Section padding: py-20 to py-32 (80px-128px)
Container max-width: max-w-7xl or max-w-6xl
Card gaps: gap-6 to gap-8
```
